## 🚀 Mini Projetos

### 🧮 Calculadora de IMC 🚧

Aplicação em Python que calcula o Índice de Massa Corporal (IMC) com foco em lógica, validação de dados e evolução contínua.

---

### 🧠 Objetivo

Evoluir de um exercício simples para um projeto mais completo, aplicando boas práticas de programação.

---

### ✅ Já implementado

* Lógica simplificada com condições mais claras (`< 18.5`, `< 25`, `< 30`)
* Validação de entrada (peso e altura maiores que 0)
* Cálculo do IMC
* Exibição com 2 casas decimais
* Classificação:

  * Abaixo do peso
  * Peso ideal
  * Sobrepeso
  * Obesidade
* Mensagens mais descritivas para o usuário

---

### 🚧 Em desenvolvimento

* Cálculo de peso ideal (mínimo e máximo)
* Recomendações personalizadas (quanto ganhar ou perder peso)
* Loop para múltiplos cálculos
* Menu interativo
* Histórico de dados (últimos cálculos)

---

### 🛠️ Tecnologias

* Python

---

### ▶️ Como executar o projeto

1. Clone o repositório:

```
git clone https://github.com/gabriellechristie-dev/mini-projetos.git
```

2. Acesse a pasta do projeto:

```
cd mini-projetos/calculadora-imc
```

3. Execute o arquivo:

```
python calculadora_imc.py
```
